The whole idea is to kickstart an Electron application, by not bothering

- Setting up Auto-Updates
- Configuring Notarization
- Adding Tray functionality

### Included

- Typescript
- React
- React-Router v6
- Chakra UI
- Immer

### Install

```sh
npm install
```

### Usage

```sh
npm run start
```

### Publish

Edit `dev.app-update.yml` & `electron-builder` and run:

```sh
npm run release
```
